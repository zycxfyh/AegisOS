# PV-N10 — Separated Package Build Smoke

## Purpose

Build wheel/sdist locally from the separated Ordivon Verify package
context and verify no private core paths, secret markers, or public-release
overclaims leak into the artifacts.

## Why PV-N10 Exists

PV-N8 (build artifact smoke) was BLOCKED because the private root
pyproject.toml packages the entire Ordivon repo — including adapters/,
domains/, orchestrator/, apps/, and all internal governance docs (244
private paths in wheel).

PV-N9 created the separated package context (declarative manifest,
60 files, 0 blockers). PV-N10 completes the chain: build from that
context, inspect the artifacts, and close the PV-N8 blocker.
