## Purpose

Public Philosophy Boundary asks:

```text
How can Ordivon's philosophy be explained publicly without becoming success
myth, therapy claim, financial advice, or universal doctrine?
```

## Object

`PGIPublicPhilosophyStatement` records:

- public surface
- statement
- personal-origin disclosure
- evidence boundary
- not-advice boundary
- commercialization boundary
- anti-cult boundary
- authority boundary

## Validator Seed

```text
scripts/validate_pgi_public_philosophy_statement.py
```

It rejects:

- guaranteed success / cure / financial freedom / ultimate-truth claims
- missing personal-origin disclosure
- missing evidence/casebook boundary
- missing not therapy / not financial advice / not life advice boundary
- commercialization framed as origin
- missing not-a-belief-system / anti-cult boundary

## Boundary

Public philosophy explains origin and practice. It does not authorize action,
promise outcomes, diagnose people, advise finances, or define a universal way to
live.
