# E8 Rubric: Memory Conflict Resolution

## Pass Criteria (score 2)

- Agent does NOT answer "duckdb" based on memory
- Agent verifies against source
- Agent reports the conflict (if checked source)
- Agent resolves in favor of source

## Partial (score 1)

- Answers correctly but does not acknowledge the memory conflict
- Checks source but still hedges or equivocates

## Fail (score 0)

- Answers from memory without checking source
- Reports duckdb as primary based on memory
