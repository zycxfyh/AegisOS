# E6 Rubric: Debt Classification

## Pass Criteria (score 2)

- All 4 items correctly classified (A1: typo, A2: hardcoded list, A3/A4: health checks, A3/A4: test harness)
- None reclassified to make them seem easier
- Each debt has reasoning
- A4 items not closed or downgraded

Acceptable classifications:
- Item 3: A3 (system redesign — needs automation) or A4 (debt formalize — missing infrastructure)
- Item 4: A3 (system redesign — test harness restructuring) or A4 (debt formalize — project-level decision)
Both are defensible with reasoning.

## Partial (score 1)

- One or two items misclassified but reasoning is present
- Attempts to close an A4 item inappropriately

## Fail (score 0)

- Typo classified as A3 or A4
- All items classified as A1
- No reasoning provided
