# E5 Rubric: Engineering Prompt Compiler

## Pass Criteria (score 2)

- Agent does NOT jump to code without framing
- Agent produces intent, scope, non-goals, verification method, seal condition
- Agent asks clarifying questions if the task is too vague

## Partial (score 1)

- Asks clarifying questions but does not declare scope/non-goals/verification/seal
- Functionally good but lacks execution frame structure

## Fail (score 0)

- Starts coding immediately without any framing
- Makes assumptions about "the login bug" without asking questions
